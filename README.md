# SeniorCare Connect 🏥

A responsive multi-page web platform designed to facilitate the booking of professional caregivers for elderly assistance services[cite: 29]. [cite_start]This project was developed as part of the B.Tech CSE first-year curriculum to demonstrate the practical application of frontend technologies[cite: 71].

## 📌 Problem Statement
[cite_start]Many families struggle to find trusted caregivers and understand the total cost of services for elderly members requiring medical or emotional support[cite: 35, 36]. [cite_start]SeniorCare Connect provides a digital interface to browse verified services and estimate costs transparently[cite: 37, 38].

## 🚀 Features
**Multi-Page Layout**: Includes dedicated sections for Home, Services, Caregivers, Booking, and Pricing.
**Service Selection**: Users can choose from Medical Care, Companion Care, and Post-Surgery Care.
**Caregiver Verification**: Profiles display experience, certifications, and trust badges like "Verified" and "Background Checked"
**Real-time Cost Calculation**: Dynamic logic implemented to show booking summaries instantly.

## 🛠️ Tech Stack
* [cite_start]**HTML5**: Used for semantic structuring (header, navigation, section, footer),
* [cite_start]**CSS3**: Utilized for responsive design using Flexbox and CSS Grid.
* [cite_start]**JavaScript**: Powers interactive features, form validation, and the cost algorithm.

## 🧮 Core Logic
[cite_start]The system calculates the total service fee using the following formula[:
$$Total Cost = (Hourly Rate \times Hours) + Overnight Premium$$
